===================
Print square module
===================

This module defines the ``print_square(size)`` function.

Usage
======

``print_square(...)`` prints a square using the # character.

::

===================

    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(0)

    ######
    ######

    >>> print_square(4)
